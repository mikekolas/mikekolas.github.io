TCL - Wireless tachometer

This is the first team project I worked on. Along with my friend Stelro he had the task to implement a real time system to measure the RPM of a single fan and get graphical representation. After the first version, a new task presented. To implement the system for 15 fans with the use of a CD74HC4067 multiplexer. I implemented the hardware changes and installation of 15 sensors, while Stelro worked on the new GUI with Qt Framework and C++.

TCL's operation

The system uses infrared sensors to be installed near the calculated fan-turbines. On every round, the system calculates the passes of a single second and by multiplication it finds the RPM. There are different codes on TCL repository, giving examples of the possibillities.
